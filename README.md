# Project Description
The Library Management System is a beginner-friendly SQL project designed to manage library operations efficiently. The database stores information about books, library members, and borrowing records.
This project demonstrates fundamental database concepts such as table creation, primary keys, foreign keys, relationships between tables, and SQL queries. It also includes INNER JOIN operations to retrieve borrowing information by combining data from multiple tables.

# Features
- Manage book inventory
- Store member information
- Track borrowed and returned books
- Retrieve borrowing history using JOINs
- Practice real-world database relationships

# Skills Demonstrated
- SQL Database Design
- Table Creation
- Primary Keys & Foreign Keys
- Data Insertion
- INNER JOIN Queries
- Data Retrieval and Analysis

# Tools Used
- SQLite
- DB Browser for SQLite
